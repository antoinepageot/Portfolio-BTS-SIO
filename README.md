# Portfolio BTS SIO - Antoine Pageot

Site statique pour le portfolio. Ouvrir `index.html` dans un navigateur ou utiliser une extension Live Server.

Structure recommandée :

- `index.html`, `veille.html` — pages principales
- `css/main.css` — styles principaux
- `assets/images/` — images
- `docs/` — documents PDF

Pour développement local rapide (VS Code) : installez "Live Server" puis ouvrez `index.html`.
